## 🚀 About

Nefertiti is a modern web application built with **TypeScript** as the primary language, with styled components using **CSS** and minimal **JavaScript** integration.

## 📁 Project Structure

This project uses **file-based routing** with TanStack Start. Each `.tsx` file in the routes directory automatically becomes a route.

## ✨ Features

- ⚡ TypeScript for type-safe development
- 🎨 CSS for styling
- 📦 Modern tooling and best practices
- 🔄 File-based routing system

## 🔧 Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```
